Usando a class table na tabela organiza automaticamente todos os dados com espaçamentos e linhas

Usando a class table-dark na tabela inverte as cores deixando o fundo escuro e os dados em branco

Usando a class table-striped na tabela conseguimos diferenciar as linhas mudando altomaticamente o background das linhas

Usando a class table-bordered na tabela conseguimos deixar a tabela com bordas ao redor e entre as linhas

Usando a class table-hover na tabela conseguimos deixar a tabela com efeitos de hover sobre as linhas

Usando a tag thead para separar o cabeçalho da tabela e a class tbody no corpo da tabela conseguimos estilizar melhor a tabela

Usando a class thead-dark na tag thead deixamos o cabeçalho da tabela com o fundo dark, usando a class thead-light conseguimos um deistaque cinza

Usando a class table-sm conseguimos diminuir os espaçamentos na tabela
